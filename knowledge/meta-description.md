# Meta Description Deep Guidance

Meta descriptions do not directly influence rankings, but they directly influence click-through rate from the search results page — which is a strong indirect ranking signal over time.

## Structure that converts
1. Lead with the core value or answer (what the page delivers).
2. Include the focus keyword once, in a natural position.
3. Add a specific, credible detail (a number, a timeframe, a named benefit) — specifics outperform generic claims.
4. Close with a soft call to action only if it fits naturally ("See how", "Compare options") — never a hard sales pitch.

## Length discipline
- Hard target: 140–160 characters including spaces.
- Google rewrites descriptions it judges to be low-quality or irrelevant to the query, so relevance to the page's actual content matters more than keyword density.

## What to avoid
- Duplicating the SEO title word-for-word.
- Starting every description across a site with the same template phrase ("Learn more about...") — this reads as templated and hurts CTR at scale.
- Keyword stuffing or unnatural repetition.
- Promises the page content does not fulfill (a common cause of high bounce rate even when CTR is good).

## For pages without a natural single focus (categories, archives)
Summarize what the visitor will find in this collection and why it's organized this way, rather than trying to force a single-page pitch.
